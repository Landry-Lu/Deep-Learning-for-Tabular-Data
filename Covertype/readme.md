## Dataset

The dataset used in this project is the **Covertype dataset**, a large-scale tabular dataset for multi-class classification.

Due to GitHub file size limitations (25MB per file), the dataset is **not included in this repository**. Instead, it can be downloaded from the original course webpage provided by the instructor:

https://alexxthiery.github.io/teaching/DSS5104/DL-for-tabular-data.html#key-questions-to-address

Please follow the instructions on the page to obtain the dataset.

## Dataset Description

The Covertype dataset contains approximately 581,000 samples with 54 numerical features. The task is to classify forest cover types into 7 categories.

## Usage

After downloading the dataset, place the data file in the appropriate directory (e.g., `data/`) and update the file path in the notebook if necessary.

## Note

This project follows the dataset usage guidelines provided in the course materials. The dataset is used strictly for educational and research purposes.
